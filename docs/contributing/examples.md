# Examples

These packs are good starting points when you want to understand tone, structure, and portability:

- [Liquid Glass Enforcer](/skills/liquid-glass-enforcer)
- [RAG System Architect](/skills/rag-system-architect)
- [Prompt Injection Firebreak](/skills/prompt-injection-firebreak)
- [Cost Router Tuner](/skills/cost-router-tuner)
- [Failure Replay Lab](/skills/failure-replay-lab)

When you study examples, look for:

- a clear purpose line
- realistic trigger signals
- compatible client surfaces
- install and export commands that match the repo
- related skills that make composition easier

# Privacy

The SkillForge docs site is a static GitHub Pages deployment.

- The site itself does not include a custom user account system.
- The local search index runs in the browser.
- External services such as GitHub may collect their own data when you follow outbound links.
- If this changes, this page should change with it.

For the repository itself, see the public source and documentation in
[GitHub](https://github.com/jamiojala/skillforge).
